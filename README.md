# RE-DRAFT

## 作品名

流出原稿（仮）／RE-DRAFT

## 概要

Googleドキュメント風の共同編集画面で、作者がリアルタイムに小説を書き進める様子を見守るWEBゲーム。  
プレイヤーは「流出した共有リンク」から原稿を覗き見し、異変を見つけたら「指摘する」ボタンを押す。  
序盤は笑えるミス直しゲームだが、段階が進むにつれて作者の精神状態の崩れが原稿に滲み出し、プレイヤーは「指摘が救済なのか加害なのか」迷い始める。

## 表向きのWeb形式

Googleドキュメント風の共同編集ドキュメント（1ページ完結WEBゲーム）

## 制作目的

ARG（代替現実ゲーム）コンテンツ。  
「変な原稿を直すゲーム」が「作者の崩壊を見守るゲーム」に変わっていく体験を届ける。

## 管理ファイル一覧

| ファイル | 内容 |
|---|---|
| brief.md | 企画概要（完成） |
| research.md | 設定リサーチ・参考資料 |
| scenario.md | シナリオ本文・異変テキスト・コメント文 |
| arg-design.md | ARGギミック設計・到達方法 |
| implementation-spec.md | 実装仕様（技術詳細） |
| asset-list.md | 素材リスト |
| review-log.md | レビュー記録 |
| redraft_企画概要.md | 原案：企画仕様書 |
| redradt_開発仕様書.md | 原案：技術仕様書 |
| redraft_ゲーム内小説.txt | 原案：ゲーム内小説完成稿 |

## 制作フロー

```
/new-project（完了）
↓
/scenario　← 次はここ
↓
/arg-design
↓
/implementation-spec
↓
/implement
↓
/review
```
